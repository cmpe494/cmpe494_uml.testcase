```plantuml
@startuml
!theme spacelab

title Sequence Diagram: Generating a Filtered Financial Report (UC-D5)
autonumber

actor "User" as User
participant "Frontend (UI)" as Frontend
participant "Backend (API)" as Backend
database "Database (DB)" as DB

User -> Frontend: Accesses "Reports" screen

group Report Filtering
    User -> Frontend: Selects report type "Daily Sales"
    activate Frontend
    Frontend --> User: Updates UI with selection
    deactivate Frontend

    User -> Frontend: Sets date range "Last 7 Days"
    activate Frontend
    Frontend --> User: Updates UI with date range
    deactivate Frontend

    User -> Frontend: Clicks "Generate" button
end

activate Frontend
Frontend -> Backend: Request filtered report (type: Daily Sales, range: Last 7 Days)
activate Backend
Backend -> DB: Query records for Daily Sales within Last 7 Days
activate DB
DB --> Backend: Return matching records
deactivate DB

Backend --> Frontend: Send report data
deactivate Backend
Frontend -> User: Display report in table format
deactivate Frontend

@enduml
```
