```plantuml
@startuml
!theme mars

actor Player
participant GameClient
participant BackendSettingsService as Backend
participant LocalConfigurationFile as LocalFile

title In-Game Settings Persistence (UC-G6)

Player -> GameClient: Changes a game setting (e.g., Volume to 80%)
activate GameClient

alt Backend is reachable (Basic Flow: User changes settings)
    GameClient -> GameClient: Apply setting immediately (local state update)
    GameClient -> Backend: Send new configuration
    activate Backend
    Backend --> GameClient: Confirm storage
    deactivate Backend
else Backend is unreachable (Exceptional Flow I1: Sync settings later if backend is down)
    note right of GameClient: Attempting to connect to Backend...
    GameClient ->x Backend: Attempt to send configuration (fails)
    note right of GameClient: Backend service is unreachable.
    GameClient -> LocalFile: Save setting to local configuration file
    GameClient -> GameClient: Mark setting for later synchronization

    group Automatic Synchronization when Online
        note over GameClient, Backend: Later, when Backend service becomes available...
        GameClient -> Backend: Sync locally saved settings
        activate Backend
        Backend --> GameClient: Confirmation of synchronization
        deactivate Backend
    end
end

deactivate GameClient

@enduml
```
