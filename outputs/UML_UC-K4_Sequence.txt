```plantuml
@startuml
!pragma layout smetana

actor User as "User"
participant Kiosk as "Kiosk"
participant GameClient as "GameClient"

title Game Initialization (UC-K4)

User -> Kiosk: Clicks "Start" button

alt User starts the game successfully
    activate Kiosk
    Kiosk -> GameClient: "start session" command
    activate GameClient
    GameClient --> Kiosk: Acknowledges command
    deactivate GameClient
    Kiosk -> Kiosk: Switches to "Game In Progress" screen
    deactivate Kiosk
else GameClient fails to start
    activate Kiosk
    Kiosk -> GameClient: "start session" command
    note right of Kiosk: Kiosk waits for response within timeout period
    Kiosk -[dashed]> GameClient: <size:10><i>(Timeout)</i>
    GameClient --x Kiosk: <color:red>No response
    Kiosk -> Kiosk: Displays "Game could not be started" error message
    Kiosk -> Kiosk: Triggers refund or retry mechanism
    deactivate Kiosk
end

@enduml
```
