```plantuml
@startuml
!theme mars

actor User
participant KioskClient
participant Backend
participant "Offline Communication Strategy (UC-C5)" as OfflineStrategy

' Background: Given the KioskClient has an active low-latency connection to Backend
' (This is a precondition for the basic flow and is assumed, not an interaction step.)

group Basic Flow: Successful real-time permission check
    User -> KioskClient: Scans card/QR
    activate KioskClient
    KioskClient -> Backend: Validation Request (Card/QR Data)
    activate Backend
    note right of Backend: Verify balance and permissions
    Backend --> KioskClient: "Access Granted"
    deactivate Backend
    note left of KioskClient: Response within 500 milliseconds
    KioskClient -> KioskClient: Activate game session
    deactivate KioskClient
end group

group Alternatif Akış A1: Backend denies request due to insufficient funds
    KioskClient -> Backend: Session Start Request
    activate KioskClient
    activate Backend
    note right of Backend: User wallet has insufficient funds
    Backend --> KioskClient: "Insufficient Balance"
    deactivate Backend
    KioskClient -> User: Display "Please Top-up" message
    deactivate KioskClient
end group

group İstisna Akış I2: Connection loss triggers offline mode
    KioskClient -> Backend: Attempt to send request
    activate KioskClient
    KioskClient -[#red]X Backend: Connection Lost!
    note over KioskClient: Detects connection loss
    KioskClient -> OfflineStrategy: Switch to Offline Communication Strategy
    activate OfflineStrategy
    KioskClient -> KioskClient: Log connection failure
    deactivate OfflineStrategy
    deactivate KioskClient
end group

@enduml
```
