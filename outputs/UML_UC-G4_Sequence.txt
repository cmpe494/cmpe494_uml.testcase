```plantuml
@startuml
' Define participants in the interaction
participant GameClient
participant Network
participant GameServer
participant LocalStorage

' Background: The game is running, and the network connection has been lost.
' The sequence starts with GameClient actively monitoring and detecting this loss.
GameClient -> Network: Monitor Connection Status
Network --x GameClient: Connection Lost!

' Basic Flow: Detect loss and successfully reconnect
GameClient -> GameClient: Detects disconnection (When)

loop Reconnect attempts (every 5 seconds)
    GameClient -> Network: Attempt to Reconnect
    activate Network
    alt Connection Re-established (Successful reconnect path)
        Network --o GameClient: Connection Re-established (When)
        deactivate Network
        note right of GameClient: Reconnection successful. Exiting reconnect loop.
        break loop
    else Reconnect Failed for this attempt
        Network --x GameClient: Connection Failed / Timeout
        deactivate Network
        note right of GameClient: Wait 5 seconds before next attempt
    end
end

' After the reconnect loop, evaluate the outcome
alt Connection was successfully re-established (Scenario: Detect loss and successfully reconnect)
    note over GameClient: Connection has been re-established.
    GameClient -> GameServer: Flush "Local Telemetry Queue" (Then)
    activate GameServer
    GameServer --> GameClient: Telemetry Flushed Acknowledgement
    deactivate GameServer
    GameClient -> GameClient: Continue normal operation (And)
else Max reconnect attempts failed (Scenario: Permanent connection loss)
    note over GameClient: GameClient attempts to reconnect multiple times without success (When)
    note over GameClient: Maximum reconnect attempts reached or deemed permanent.
    GameClient -> GameClient: Continue game until the end (Then)
    GameClient -> LocalStorage: Save final score
    activate LocalStorage
    LocalStorage -> LocalStorage: Mark final score for "Sync on Next Boot" (And)
    LocalStorage --> GameClient: Score saved locally
    deactivate LocalStorage
end

@enduml
```
