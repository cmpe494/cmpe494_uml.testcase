```plantuml
@startuml
!theme superhero
title Highscore Display System Interaction Flows

actor "Player" as PlayerObserver #lightblue
participant "Highscore Display System" as HDS #LightGreen
participant "Configuration Store" as Config #LightGrey
participant "Backend Score Service" as BSS #LightBlue
database "Local Cache" as Cache #LightSalmon
participant "Display UI" as UI #Orange

autonumber

PlayerObserver -[hidden]-> HDS : //(Player implicitly views the display)//

note over HDS : System startup and initialization (Background)
HDS -> Config : Read configured scope (e.g., Global/City/Zone)
activate HDS
Config --> HDS : Return "<scope>"
deactivate HDS

loop Auto-refresh / Periodic Update (e.g., every 30 seconds)
    activate HDS
    HDS -> BSS : requestScoreList(scope)
    activate BSS

    alt Backend Score Service is reachable and returns data (Basic Flow: "Display scores for defined scope")
        BSS --> HDS : returnSortedScoreList(scores)
        deactivate BSS

        HDS -> Cache : storeScores(scores)
        activate Cache
        Cache --> HDS : Scores stored successfully
        deactivate Cache

        HDS -> UI : displayScores(scores)
        activate UI
        UI --> HDS : Scores displayed
        deactivate UI

        note right of HDS: Auto-refresh timer for 30 seconds is set

    else Backend Score Service is unreachable (Alternative Flow A1: "Display cached data when backend is down")
        BSS --[#RED]> HDS : Connection refused / Timeout
        deactivate BSS

        HDS -> Cache : retrieveLastValidScores()
        activate Cache
        Cache --> HDS : returnCachedScores(cachedScores)
        deactivate Cache

        HDS -> UI : displayScores(cachedScores)
        activate UI
        HDS -> UI : displayIndicator("Offline")
        UI --> HDS : Cached scores and "Offline" indicator shown
        deactivate UI

        note right of HDS: Continues periodic update attempts

    else Backend Score Service returns empty list (Alternative Flow A2: "No scores available for the scope")
        BSS --> HDS : returnEmptyScoreList()
        deactivate BSS

        HDS -> UI : displayMessage("No scores yet")
        activate UI
        UI --> HDS : "No scores yet" placeholder displayed
        deactivate UI

        note right of HDS: Continues periodic update attempts
    end
    deactivate HDS
end loop

@enduml
```
